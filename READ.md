# Smart Order Management System v2

## Overview
This project analyzes simple e-commerce order data using Python collections and conditional logic.

## Topics Covered
- Lists
- Dictionaries
- Sets
- Tuples
- Boolean values
- If / Else

## Features
- Total order analysis
- Paid and unpaid order tracking
- Unique customer extraction
- Product analysis
- Order summaries

## How to Run

```bash
python Smart_order_management_day2.py
