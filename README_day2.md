# Day 02 - Smart Order Management System

## Overview

This project analyzes simple e-commerce order data using Python collections and conditional logic.

## Project Goal

The goal is to model and analyze order data in a realistic e-commerce scenario.

## Topics Covered

- Lists
- Dictionaries
- Sets
- Tuples
- Boolean values
- Comparisons
- If / elif / else statements

## Features

- Counts total orders
- Counts paid and unpaid orders
- Extracts customer names
- Removes duplicate customers using set
- Analyzes ordered products
- Detects repeated products
- Creates order summaries using tuples

## Example Output

```text
Total orders: 3
Paid orders: 2
Unpaid orders: 1

Unique products: {'Laptop', 'Mouse', 'Keyboard', 'Monitor', 'HDMI Cable'}
Order summaries: [('Ahmet', 18500), ('Zeynep', 2500), ('Mehmet', 7200)]

## How to Run

```bash
python smart_user_data_cleaner_day2.py
```
