# Day 03 - Smart Survey Analyzer v3

## Overview
This project analyzes university student satisfaction survey results using Python loops, control flow, and analytical logic.

## Project Goal
The goal of this project is to simulate and process survey data while developing strong foundations in loops, decision-making, and data analysis.

## Topics Covered
- For loop
- While loop
- Break / Continue / Pass
- sum, len, min, max, round
- random

## Features
- Generates random survey scores (1–5)
- Stops when 0 is entered
- Skips invalid scores (6)
- Calculates average, minimum, and maximum
- Categorizes responses:
  - Satisfied
  - Neutral
  - Unsatisfied
- Detects critical dissatisfaction patterns
